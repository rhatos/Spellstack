## Installation

### Prerequisites
1. [Unity Hub](https://unity.com/download) must be installed
2. Unity 6 (`6000.3.1.10f1`) must be installed — follow the [Unity install guide](https://learn.unity.com/pathway/junior-programmer/unit/player-control/tutorial/install-unity-software?version=6.0) if needed
3. Git must be set up on your device, or [GitHub Desktop](https://desktop.github.com/) should be installed and signed in

### Instructions
1. Clone the repository:
```
   git clone https://github.com/rhatos/Spellstack.git
```
2. Open Unity Hub and navigate to the cloned repo's location
3. Open the project — it may take some time to compile all assets and scripts

---

## Running the Game

Press the **Play** button above the Scene/Game view in the Unity Editor.

Ensure that the AudioManager is within the shared scene list.

Any issues with the game that occur please email us, thank you.

## Implementation

### Tutorials
The Audio Manager system was implemented using a [Brackeys tutorial](https://www.youtube.com/watch?v=6OT43pvUyfY).

### Generative AI
- **Gemini** was used for asset generation of some spell combos: Burning Vines, Bog Trap, Mudslide, Vine Cyclone, and Steam Explosion
- **Gemini** was also used to generate code for navigating between the main menu, story overview, and instruction screen

### Assets & Audio
- 2D spell and character/mob sprites were sourced from [itch.io](https://itch.io), with some adjusted to fit the game's aesthetic using [Libresprite](https://libresprite.github.io/)
- Sounds were sourced from [Pixabay](https://pixabay.com/) and Storyblocks; some 8-bit sounds were created using [bfxr](https://www.bfxr.net/)
- Music was composed in [Reaper DAW](https://www.reaper.fm/) using built-in synthesisers

### Code
The codebase was written by the team, with minor AI assistance for small changes.
